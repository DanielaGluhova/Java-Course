JVM - Java Virtual Machine - Когато казваме че Java e платформено независима
 е точно заради JVM. На всяко устройство в зависимост от ОС има JVM за да могат 
 да се изпълняват Java програми. Работата на JVM е да изпълнява програми. JVM не приема кода директно... 

Когато ние като програмисти напишем кода на програмен език
 създаваме файл със съответното разширение(.java). След това компилатора(javac) го чете и превежда
 и го преобразува в байт код с разширение(.class). Тогава вече JVM ще може да изпълни.
Програмата ни може да има много файлове, но JVM ще изпълни ще стартира с този който има метода main.

JRE - Java Runtime Environment - среда която работи  с JVM и библиотеки. Като програмисти
 ние изтегляме JDK вътре има JRE вътре има JVM.
