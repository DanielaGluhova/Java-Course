
Java Naming Convention
Използва се Camel(myMethod) casing:
* Интерфейси (Interface) - започват с ГЛАВНА БУКВА -> Runable
* Класове (Class) - започват с ГЛАВНА БУКВА -> Calculator
* Методи (Method) - започват с малка буква -> main
* Полета (Field) - започват с малка буква -> name
* Константи (Constant) - изписват се с ГЛАВНИ БУКВИ -> PI, BRAND
* Конструктори (Constructor) - изписват се като класове но задължително имат 
  скобите на метод - () -> Human()
